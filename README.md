# Lumière Gallery — Premium Photo Experience

Lumière Gallery is a stunning, high-performance, premium web-based image gallery featuring beautiful visual components, smooth micro-animations, filters, search functionality, and a rich lightbox preview with real-time filters (Black & White, Sepia, Warm, Cool, Vivid, and Fade).

This project showcases clean semantic HTML5, highly responsive modern CSS styling (with modern typography, custom gradients, CSS variables, and fluid grids), and vanilla JavaScript.

## Features

- **Dynamic Interactive Design:** Beautiful mouse glow background effect, page scroll progress bar, and floating decorative ambient shapes.
- **Filtering & Search:** Real-time category-based filtering (All, Nature, Urban, Wildlife, Architecture) and live keyword search.
- **Premium Lightbox View:**
  - Fullscreen high-resolution viewing.
  - Responsive navigation (Prev/Next buttons, keyboard navigation with Arrow keys and Escape).
  - Built-in photo gallery strip for quick navigation.
  - Interactive real-time CSS image filters (Grayscale, Sepia, Warm, Cool, Vivid, and Fade).
- **Responsive Layout:** Grid dynamically shifts sizes gracefully across mobile, tablet, and desktop screens.
- **Accessibility (a11y) & SEO:** Built using modern semantic HTML, keyboard accessible elements, `aria-*` tags, and proper alt descriptions.

## Technology Stack

- **HTML5:** Structuring and layout.
- **CSS3:** Custom properties (CSS variables), grid, flexbox, glassmorphism UI style, keyframe animations, and filter effects.
- **JavaScript (ES6):** Dynamic content rendering, filter operations, lightbox navigation, state management, and interaction handlers.

## Getting Started

To run this project locally, simply clone the repository and open the index file:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/bharathbalikant-hue/CodeAlpha_-image-gallery.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd CodeAlpha_-image-gallery
   ```
3. **Open the project:**
   Open `bharath 1/index.html` in any web browser.
